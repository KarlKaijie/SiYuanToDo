# ToDo List
## 动态更新的ToDo List

## 缘由

> 本来用着Notion做笔记，转到思源后，发现没有动态更新的Task List了，市场里也只有一些静态更新的模板。思量着，那就自个做一个吧。 **位置: 集市 -> 挂件-> SiYuanToDo**

## 介绍
* 支持和思源文档和块关联
* 支持筛选
* 每个文档都可以建一个todoList
* 数据默认存放在文档的目录，跟随文档跨端跨设备同步

## 其它
* 挂件初始化失败，请在引入挂件后，关闭当前文档，再打开
