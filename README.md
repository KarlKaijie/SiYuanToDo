# ToDo List
## 动态更新的ToDo List

> 如果觉得好用，还希望各位可以在github上多多关注～

## 介绍
* 支持筛选
* 每个文档都可以建一个todoList
* 数据默认存放在文档的目录，跟随文档跨端跨设备同步
